B) BENCHMARK

 

1. Portfolio return components

A portfolio’s return can be broken down into three components.

P = M + S + A

M = Market return (General Benchmark return)-Like SnP 500

B=Manager's Benchmark (SnP Growth100)

S = style

A = active management

P = portfolio return

 

Style

S= B – M 

Style which is the difference between a manager’s benchmark style and market return.

Active

A =  P – B

Active return which is the difference between portfolio return and a manager’s benchmark style.

 

2.     Properties of a valid Benchmark (SAMURAI)

   Specified in advance: known to both the investment manager and fund sponsor

   Appropriate: consistent with the investment manager’s approach and style

   Measurable: readily calculable

   Unambiguous: clearly defined identities and weights of securities     

   Reflective of the current management’s opinion: Manager has current knowledge

   Accountable: Manager should accept accountabilities

    Investable: possible to replicate the benchmark and forgo investment management.

 

 

3.     Types of benchmarks

Absolute: a return objective

Advantage: 1) simple   2) specified in advance

Disadvantage:  1) Not investable

 

Manager Universe:

Advantage – Measurable

Disadvantage -

1.       Subject to survivorship bias (underperforming manager’s performance dropped)

2.       Can’t be identified and specified

3.       Not investable

 

Broad market indices

Advantage: 1) well recognized and easy to understand

Disadvantage: 1) The manager’s style may deviate from the style in the index.

 

Factor Model

Advantage: 1) useful in performance evaluation

  2) capture factor exposures

Disadvantage:    1) Not intuitive

2) May be ambiguous (different factor models can produce different output)

 

Return based:

Advantage: 1) Generally easy to use and intuitive

Disadvantage      1) Enough monthly returns would be necessary

2) The style indices may not reflect what the manager owns

 

Custom security based:

In a liability-based benchmark component weights are determined based on the requirement that the benchmark closely track returns to the liabilities.

 

 

Advantage: 1) Meets all the required benchmark criteria 

Disadvantage: 1) expensive to construct 2) a lack of transparency by the manager

 

 

4. Benchmark Quality:

A. Systematic bias – close to beta 1 is desirable

B. Tracking error – lower, better

C. Risk characteristics – similar risk characteristics as portfolio is better

D. Coverage – the higher coverage, the better (market value of securities are both in the benchmark and portfolio)

E. Turnover – lower better (caused by frequent buy & sell)

F. Positive active position – Higher better (lower active position implies that the benchmark does not reflect the manager’s process and not appropriate).

 

5.     Hedge Fund Benchmark

 

Hedge Fund Benchmark issues

 

a.       Initial investment can be insignificant (v0=small) so difficult to calculate return for comparison.

b.       Frequent trade in long/short strategy makes assets composition very dynamic, which makes it difficult to calculate return.

c.       Some hedge fund target an absolute return which makes other benchmarks irrelevant.

d.       Only same style of hedge fund can be compared, different style can't be compared.

e.       Since hedge funds returns are skewed so measures like standard deviation and sharp ratio usage is questionable.

 

Hedge fund benchmark alternative:

 

a.       Separate bench mark for long and short.

b.       Since V0 (denominator can be low or zero) so only calculate the value added = V1-V0

c.       Compare sharp ratio of similar style hedge fund.

 

 

6.     Limitation of SnP as a Benchmark

 

1.       Large cap bias

2.       Double counting results from corporate cross ownership.

3.       Inclusion of some international (Non US Based) stocks.

4.       Failure to cover 100% US stocks (excludes some OTC stocks/AMEX stocks). It includes on NYSE stocks.

 

 

Points

 

1.       Owned Benchmark indicates that the investment manager should be aware of and accept the responsibility for the constituents and performance of the benchmark.

2.       Benchmark should be rebalanced at predetermined intervals.
