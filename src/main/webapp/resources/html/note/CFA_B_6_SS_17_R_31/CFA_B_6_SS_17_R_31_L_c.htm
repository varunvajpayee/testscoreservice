2. Micro Performance Attribution

 

Difference in the returns between portfolio and benchmark = Different due to weights + different due to different security selection

{Pure Sector Allocation}    {With Sector selection + Allocation interaction }

 

P=Manager's portfolio

B=Benchmark

Wp,i = weights of manager's portfolio in sector i.

Wb,i = weights of benchmark in sector i.

 

Pure Sector Allocation - Sector weights differ but with-in sector same securities are used. This portion of the return is attributable to sector weights difference.

 

Within-Sector Selection- Sectors weights are the same but within the sector the manager may hold securities in different-from-benchmark weights. This portion of the return is attributable to Security selection.

 

Allocation(Selection) Interaction- This return should be close to 0, if benchmark is good. It involves the joint effect of security selection and portfolio. It's kind of catch all, returns which cannot be explained by with pure section allocation and within-sector selection falls into this category.
