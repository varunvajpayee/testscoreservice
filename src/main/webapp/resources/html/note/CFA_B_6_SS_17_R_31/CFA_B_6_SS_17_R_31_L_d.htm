D) FIXED INCOME PERFORMANCE ATTRIBUTION

 

1.     Return due to external environment

a.      Expected Interest Rate Effect

Benchmark return if interest rate has moved as per the forward curve predicted.

b.      Unexpected Interest Rate Effect

Benchmark return after incorporating the actual interest rate changes. It doesn't consider any actions taken by the manager.

1.     Return Contribution (Value Added) by manager

a.      Interest rate management

Return attributed to the manager's selection of portfolio duration, convexity and yield-curve shape changes.

Measures the manager’s ability to predict changes in interest rates and adjust the portfolio duration and convexity as per the expectation.

 

b.      Sector/Quality management

Return attributed to the manager's selection of sector and quality of the bonds. Example - if a manager sells corporate bonds and the corporate bond spread widens, the portfolio will perform better against a treasury-only portfolio.

 

c.       Security Selection

Return attributed to the actual bonds selected. Example- if manager holds apple's bond and apple's bond spreads narrow more than the IBM's bond's spread then manager's return will be better on relative basis.

 

d.      Trading Effect

 It's catch all figure. Unexplained components like transaction costs and   trading activities are covered here.
