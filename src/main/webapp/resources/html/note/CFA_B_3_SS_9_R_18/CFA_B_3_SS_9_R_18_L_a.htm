3.     Covered Vs Uncovered Interest rate Parity

 

Interest rate parity

 

It based on the assumption that there is no difference in the returns on domestic assets and the returns on foreign assets. Any interest rate differential in foreign market will have equivalent exchange rate differential, that's the reason excess returns cannot made after considering the exchange rate differences.

 

 

 

 

 by subtracting numerator with denominator.

 

 

 

Covered interest parity

 

Interest rate differential can be exploited using forward/future contracts. So, interest rate differences are covered.

 

Un-Covered interest parity

 

 it assumes, there are NO SUCH FORWARD/FUTURE contracts are available in the market, which can exploit such interest rate differentials so IR differentials are going to persist.

 

 

4.     Covered Vs Un Covered Interest rate Parity

 

RDC = RFC + RFX  (Where RDC = Investors domestic return, RFC = Foreign asset return, RFX  = Foreign currency return)

RDC = RFC + iDC - iFC (Based on above mentioned equation)

 

RDC = iDC + RFC - iFC

 

Excess return=RFC - iFC

 

 

If manager's expected return is greater than (RFC - iFC) then we should leave the exposure unhedged or

If manager's expected return is less than (RFC - iFC) then we should hedge the Currency exposure.

 

 

 

5.     Roll Yield

 

Roll yield is equal to forward premium and discount. Positive roll yield encourages hedging. Negative discourages hedging.
