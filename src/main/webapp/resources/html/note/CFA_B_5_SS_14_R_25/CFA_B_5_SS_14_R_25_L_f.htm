F) PERFORMANCE EVALUATION

 

 

1.     Standard deviation

 

Standard deviation is not a good risk measure because

 

a.       Assumes normal distribution of return.

b.       Large number of inputs required n(n-1)/2

c.       Use historical data

 

 

2.     Risk-adjusted return on invested capital (RAROC)

RAROC = Expected return/VaR

 

3.     Return over Maximum Drawdown (RoMAD)

 

Return over maximum drawdown is the average return in each year that a portfolio generates, expressed as a percentage of this drawdown figure.

 

ROMAD = Return/Maximum drawdown

 

 

4.     Sortino Ratio

 

Downside deviation computes volatility using only rate of return data points below the MAR (Minimum Acceptable Return).

 

Sortino ratio = (Mean portfolio return – MAR)/Downside deviation
