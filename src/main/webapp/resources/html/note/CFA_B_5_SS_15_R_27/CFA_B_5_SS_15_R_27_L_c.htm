2.     Interest Rate Caps, Floors, and Collars

 

Caps

Borrowers generally buy caps; A borrower receives payments at the end of each period in which the interest rate(LIBOR) exceeds the agreed strike price. So, in this way, the borrower is protected from unexpected rate increases. In exchange, the borrower must pay premium for buying the caps (calls).

 

Floors

Lenders generally buy floors; A lender receives payments at the end of each period in which the interest rate(LIBOR) falls below the agreed strike price. So, in this way, the lender is protected from unexpected rate decreases. In exchange, the lender must pay premium for buying the floors (puts).

 

Collars

Investor can create the collar by buying one position (cap or floor) and selling another position (cap or floor) so that he/she can control the interest rate fluctuations. If the premiums of these two offset each other then it's a zero-cost collar.

 

 

3.     DELTA HEDGING & SECOND ORDER GAMMA

 

Delta Hedging

 

The ratio of change between an option’s price and changes in the underlying's price. It's used to create a hedged position so that hedged position doesn't change due to changes in option or underlying prices. (Profits from options can cover underlying losses or vice versa).

 

Call Option Delta = range from 0 to 1 (Delta moved towards 1, if option is to be in the money).

Call Option Delta = range from 0 t- -1 (Delta moved towards -1, if put option is to be in the money).

 

 

Second Order Gamma

It is the rate of change of delta. It's highest when an underlying is AT the money because delta is more sensitive to the value of underlying.

It's used to gauge the volatility (riskiness) of delta.
