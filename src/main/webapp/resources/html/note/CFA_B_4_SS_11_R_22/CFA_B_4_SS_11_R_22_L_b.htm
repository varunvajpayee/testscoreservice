B) BOND RISK MEASURES

 

Risk Measures and their limitation

 

 

 

 

 

1.     Standard Deviation/Variance

Dispersion of return around the mean.

Drawbacks

1.       Normal Distribution - Bonds with embedded options have returns which are not around the mean.

2.       Number of Inputs - Large number of inputs are required to calculate the variance (N(N+1)/2) where N is number of bonds in the portfolio.

3.       Historical measures - Estimates are based on historical data. They may not represent the future.

 

2.     Semi-variance (Downside risk measure)

Dispersion of returns below a target return level.

 

Drawbacks

1.       Hard to compute

2.       Half sample size - Smaller sample size can impact the accuracy.

3.       Asymmetric returns - It's difficult to accurately forecast the downside risk when returns are asymmetric. For symmetric returns, it yields same results as Standard deviation.

 

 

3.     Shortfall Risk

Probability that the return will be less than a target rate.

 

Drawbacks

1.       Since it doesn't consider the magnitude of the outlier (only looks for probability) so shortfall risk numbers can be inaccurate.

 

 

4.     VaR (Value at Risk)

Probability that a return is less than a given threshold over a specific time (Generally mentioned as 99% Weekly VaR, 95% Monthly VaR).

 

Drawbacks

1.       It doesn't provide the maximum magnitude of loss but only provides a probability of given amount of loss.
