F) STYLE ANALYSIS

 

1.     Returns-based style analysis (RBSA)

 

·         Portfolio return is regressed again return of different indices to find out the portfolio style.

·         Weights of each coefficient tell you the portfolio's exposure to different asset classes.

·         It helps in determine the overall style of the portfolio over a date range.

 

 In principle, these indices should be:

·         mutually exclusive;

·         exhaustive with respect to the manager’s investment universe;

·         Indices should have low correlation (Different Risk factors).

 

 

𝑅p=𝑏0+𝑏1Rx1+𝑏2Rx2+𝑏3Rx3 + e

 

𝑅p = returns on manager’s portfolio

Rx1 = returns on Index1 (Say Large Cap Value)

Rx2 = returns on Index2 (Say Small Cap Value)

Rx3 = returns on Index3 (Say Large Cap Growth)

e = Error component

 

R2 (Style Fit) is the coefficient of determination, and it shows the percentage of an investor’s returns explained by the style indices.  The quantity 1 minus the style fit equals to selection, the fraction of return variation unexplained by style but explained by manager's ability to select securities.

 

2.     Holdings-based style analysis (HBSA)

Classifies individual securities by their characteristics to determine the overall style of the portfolio at a given point in time.

Generally following factors are considered: -

 

a.       Value or Growth (Determine based in P/E ratio, High P/E= Growth, Low P/E=Value, mid = Market Oriented)

b.       Expected Earnings per share growth rate (G)- High Growth rate = Growth Investor

c.       Earning Volatility (Generally Value investor invests in high volatile-cyclical stocks)

d.       Industry Representation (Growth Investor = Utilities/Financial Sector) (Value Investor = Healthcare/Technology Sector)

 

 

Returns-based style analysis VS Holdings-based style analysis

 

 

Advantages

Disadvantages

Holdings-based style analysis

Focus is on each security in the portfolio.

Does not reflect portfolio manager’s criteria security selection and portfolio design.

 

Easy (fast way) to detect a style drift at one point of time.

Involve subjectivity in picking up valuation factors.

 

 

Data Intensive

Returns-based style analysis

Focus is on each security in the portfolio.

Sometimes ineffective in determining the style

 

Easy to compare portfolios (then securities in each portfolio)

Wrong Indices in the model may lead to wrong conclusions

 

Clear theoretical basis for portfolio categorization

 

 

Less information requirement.

 

 

Low cost and fast in evaluating style drift over a period of time.
