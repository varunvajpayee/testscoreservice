H) FUNDAMENTAL LAW OF ACTIVE MANAGEMENT

 

Individual’s information ratio is the function of his/her depth of knowledge about individual securities (Information coefficient) and the number of independent bets(decision) (IB).

 

IR ≈ IC*√ BR

 

Where

IR = the information ratio,

IC = the information coefficient

IB = investor breadth (Number of Independent bets)

 

 

As per the equation, if the three are large number of bets then you need less IC (depth of knowledge) to gain information advantage (Information ratio).

 

 

Allocating to Managers

 

Once a investor decides on active management of equity, the next things, he/she needs to worry about active return, active risk and allocation of funds between active equity managers to achieve maximizing active return (while minimizing the active risk). Managing allocation to assets to managers based on risk utility function:

 

 

UA=RA−λA σA2

 

UA= Utility of active return of the manager mix

RA = expected active return of the manager mix

λA= measures risk aversion in active risk terms

σA2 = variance of the active return

 

 

Manage active investment managers

 

1.      Core satellite approach

Core holding are indexed where as active portion of the portfolio is actively managed by the manager.

2.      Completeness fund approach

A completeness fund, when added to active managers’ positions, establishes an overall portfolio with approximately the same risk exposures as the investor’s overall equity benchmark. But it may result in lower return (you end up adding some misfit securities to your fund to match the benchmark but they perform poorly). (Completeness fund acts as delta which is used to match the benchmark).

3.      Alpha and beta separation (portable alpha strategy)

Beta investment in a broad market index like S&P, for alpha, hire long-short active manager.

 

Portfolio's active return and risk

 

1.       Portfolio active return = It is a weighted average of the active returns for the individual managers.

2.       Portfolio active risk = It is a square root of the weighted sum of the variances of individual manager's return.

 

Manager's key Return and Risk types

 

1.       Manager's benchmark return (normal return) = return of correctly specified benchmark of a manager.   For value investor, it is the value index return.

2.       Investor Benchmark= Investor may use broader index for his analysis. Generally, it's used when more than one manager is used for investment.

3.       Manager’s true active return = Manager 's return − Manager’s normal benchmark

4.       Manager’s misfit active return = Manager’s normal benchmark − Investor’s benchmark

5.       Manager’s total active risk =√ (Manager’s true active risk)2+ (Manager’s misfit active risk)2

6.       Manager's Information Ratio = Manager’s true active return/Manager’s total active risk.
